Dust-lib
========


Presentation
------------

*Dust-lib* is a *typescript library* providing random helper functions for the daily life.


Links
-----

- [dust-ui](https://charlyoleg2.github.io/dust-mono/) : public instance of the UI
- [sources](https://github.com/charlyoleg2/dust-mono) : git-repository
- [pkg](https://www.npmjs.com/package/dust-lib) : dust-lib as npm-package
- [pkg-app](https://www.npmjs.com/package/dust-app) : dust-app svelte-components as npm-package
- [pkg-cli](https://www.npmjs.com/package/dust-cli) : dust-cli as npm-package
- [pkg-uis](https://www.npmjs.com/package/dust-uis) : dust-uis as npm-package


Getting started
---------------

```bash
git clone https://github.com/charlyoleg2/dust-mono
cd dust-mono
npm -w dust-lib install
npm -w dust-lib run ci
npm -w dust-lib run
```
